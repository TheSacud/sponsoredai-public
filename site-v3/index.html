How it works

Three steps. Same agent. A wallet.

Wrap & run

Install the VS Code extension, start with sai claude / sai codex, or run sai overlay both. SAI measures idle & visibility timing only: never prompts, code, or output.

Waits become cards

After a real attended wait, SAI shows one sponsor line or one small banner. At most one card every 45 seconds. Tune it or kill it.

Earnings settle

A card that stays visible ≥5s counts. The backend credits your ledger with 60% of net spend, to spend on model calls or cash out via Stripe.
